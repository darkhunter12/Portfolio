Crypto Price Dashboard

Developed a real-time cryptocurrency price dashboard that tracks Bitcoin and Ethereum prices using the CoinGecko API. The backend, built with Node.js, fetches and stores live price data in MongoDB. The frontend, designed using HTML/CSS/JavaScript, displays up-to-date prices that auto-refresh every few seconds. Implemented RESTful API endpoints to fetch and update the data efficiently. The application is modular and easily extendable to include more coins and visual analytics in the future.

Stack: Node.js, MongoDB, HTML, CSS, JavaScript, CoinGecko API

Key Features:

  • Real-time price tracking for BTC and ETH
  • MongoDB-based historical data storage
  • REST API architecture for scalability
  • Auto-refreshing frontend display
